# minmaxadvance
minmaxadvance.c
This is an implementation of the MinMax (selection sort) like of the sort algorithm which mixed with the bubble sort. 
It is just a main idea, you might it change as you like ;).
Asymptotics [O(N*LogN),...,O(N^2)].
